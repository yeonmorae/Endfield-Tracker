# ⚡ 명조: 엔드필드 숙제 체크 리스트 (Endfield Tracker)

![License](https://img.shields.io/badge/License-MIT-green)
![Version](https://img.shields.io/badge/Version-1.0.0-blue)
![Electron](https://img.shields.io/badge/Framework-Electron-47848F?logo=electron&logoColor=white)

명조: 엔드필드 개척자분들을 위한 편리하고 미니멀한 숙제 관리 도구

(일일 업무 및 주간 업무, 이벤트 일정 관리)

---

## ✨ 주요 기능 (Key Features)

* **📅 자동 리셋**: 매일 **오전 5시**에 일간 숙제가 자동으로 초기화 (주간 숙제는 매주 월요일 리셋)
* **🖱️ 초간편 체크**: 완료한 일정을 클릭하여 빠르게 표시 가능
* **⚙️ 사용자 맞춤 설정**: 나만의 숙제를 추가하거나, 필요 없는 기본 숙제는 숨기기 가능
* **🌓 투명도 조절**: 게임 화면 위에 띄워두고 쓸 수 있도록 **창 투명도 조절** 및 **항상 위** 기능을 지원
* **💾 데이터 보존**: 프로그램을 종료해도 체크 상태와 설정값이 안전하게 저장

---

## 📥 다운로드 (Download)

프로그램을 설치할 필요 없이 바로 실행 가능한 `.exe` 파일 공유

1.  [**최신 버전 릴리즈(Releases)**](https://github.com/yeonmorae/Endfield-Tracker/releases) 페이지로 이동
2.  `Endfield-Tracker-Portable.exe` 파일을 다운로드
3.  끝!
